# 🤖 Telegram бот‑финансист

Бот показывает актуальные данные по валютам, криптовалютам, акциям, товарам и индексам, умеет отправлять уведомления и ежедневную сводку.

## Возможности
- Курсы валют (USD, EUR, CNY)
- Криптовалюты (BTC, TON, SOL, USDT)
- Российские акции (MOEX)
- Товары (нефть, золото, серебро)
- Фондовые индексы
- Уведомления о резких изменениях цен
- Пороговые алерты
- Ежедневная сводка по расписанию

## Команды
- `/start` — старт и меню
- `/help` — справка
- `/ping [IP1 IP2 ...]` — пинг серверов и задержка (avg/min/max, loss)
- `/rates` — полная сводка рынков
- `/subscribe` — подписка на уведомления
- `/unsubscribe` — отписка
- `/set_alert <АКТИВ> <ЦЕНА>` — алерт по порогу
- `/view_alerts` — текущие настройки алертов

Админские:
- `/settings` — меню настроек
- `/export_pdf` — PDF‑отчет
- `/set_daily_time HH:MM` — время сводки
- `/get_daily_settings` — текущие настройки
- `/restart_daily_job` — перезапуск сводки
- `/test_daily` — тестовая сводка
- `/check_subscribers` — статус подписчиков

## Быстрый старт
```bash
cp env.example .env
# заполните BOT_TOKEN и ADMIN_USER_ID
pip install -r requirements.txt
python admin_bot.py
```

## Переменные окружения
Минимально требуется:
- `BOT_TOKEN`
- `ADMIN_USER_ID`

Остальные ключи — опционально для расширенных источников данных.
